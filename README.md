Hope Project

A safe, pastel-inspired space to explore creativity, share kindness, and support mental health.



1. Overview

Hope Project is designed to raise awareness about mental health struggles and provide a comforting, creative digital space. Users can explore mental health resources, create custom boards like Canva, send anonymous letters, and learn about the project's mission.

The project uses pastel colors, especially lavender, and features a friendly cat mascot that guides users across all pages.


2. Structure / Pages

Homepage / Main Space
- Displays Hope Project title (hope-title.png, lavender color)
- Four pastel panels: Resources, Boards, Letters, Who We Are
- Each panel has an icon and title, clicking navigates to respective page

Resources Section (Public)
- Layout: Cards with pastel frames (title, image/icon, description, optional link)
- Features:
  - Tag filter system: simple, empathetic tags such as stress, self-care, friendship, school
  - Helpline search: user types country, displays mental health helplines
- Cat mascot appears in the corner with messages such as "Remember, you are not alone"

Boards Section (Login Required)
- Login system: username and password; optional recovery note
- Board features:
  - Create multiple boards
  - Drag-and-drop objects including images, book covers, Spotify embeds, stickers
  - Resize, rotate, and customize elements fully
  - Save boards linked to account
  - Public/private toggle with shareable link
  - Full Canva-like experience: fonts, sizes, stickers, colors, text styling, layering
- Cat mascot provides tips for creativity

Anonymous Letters Section (Public)
- Form inputs: To/For field and Message Body
- Features:
  - Submit letters anonymously (no login required)
  - Live feed of letters publicly displayed
  - Search by recipient or keyword
  - Optional categories such as friendship, encouragement
- Cat mascot provides empathy tips

Who We Are Section (Public)
- Mission statement and project description
- Our mission is to raise awareness about mental health struggles and create a safe, pastel-inspired space where people can find comfort during difficult times. Through creative expression, reflection, and gentle connection, we remind everyone that:
  - They are more than their struggles
  - Their experiences do not define their worth
  - They deserve joy, support, and self-expression
- Cat mascot introduces the project warmly
